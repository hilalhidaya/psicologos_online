Proyecto: MindCore - Clínica Psicológica Online

Asumiendo que va a usar XAMPP este sería el procedimiento para cargar la base de datos (aunque imagino que ya lo conoce igualmente lo dejo por aquí por si acaso):

1. IMPORTAR BASE DE DATOS:
- Abrir phpMyAdmin
- Crear una nueva base de datos llamada: psicologos_online
- Ir a la pestaña "Importar"
- Seleccionar el archivo: database/psicologos_online.sql

2. CONFIGURAR CONEXIÓN:
- Ajustar los datos de conexión (host, usuario, contraseña) no creo que pida contraseña pero si lo hace es 0102  

3. ABRIR EL PROYECTO:
- Colocar la carpeta en htdocs/
- Acceder desde el navegador: http://localhost/MindCore/
